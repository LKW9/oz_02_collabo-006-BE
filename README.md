# 프로젝트 가이드  

### 프로젝트 처음시작 설정  

- poetry 설치하기  

- poetry 가상환경에 패키지 설치  
   
  ```bash
  poetry update
  ```

- 장고 마이그레이트  

  ```bash
  python manage.py makemigrations
  ```

  ```bash
  python manage.py migrate
  ```

- 장고 슈퍼유저 생성  

  ```bash
  python manage.py createsuperuser
  ```
